# Flux.il - CRM לסוכניות ביטוח

מערכת CRM מתקדמת לסוכניות ביטוח, עם תמיכה בריבוי סוכניות ועתיד של AI.

## התקנה והפעלה

```bash
npm install
npm run init-db
npm run seed
npm start
```

המערכת תעלה ב־http://localhost:3000

## משתמש ברירת מחדל

- **שם משתמש:** admin-flux.il
- **סיסמה:** a12345678

## רבדים

1. **מנהל סוכנות** – הרשאות מלאות, ניהול צוות (הוספה/עריכה/מחיקה, סימון מוסתר)
2. **פקידים** – צפייה ברשימת אנשי הצוות (אנשי קשר פנימיים)
3. **לקוחות** – ללא גישה (יבואו בשלבים מאוחרים)

## מיגרציה

לעדכון מסד קיים: `npm run migrate`

## מבנה הפרויקט

```
flux.il/
├── public/          # ממשק משתמש
├── src/
│   ├── config/      # הגדרות DB
│   ├── database/    # init + seed
│   ├── middleware/  # auth
│   └── routes/      # API
└── data/            # SQLite DB
```
